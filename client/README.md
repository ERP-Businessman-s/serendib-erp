# Client — React admin app

This folder holds the ERP admin screens, built with React and Vite. Each module (Inventory, Procurement, Cutting, Sales, Billing, HR) lives in its own folder under `src`, plus the login and the dashboard.

The React app is built into static files and served by the Express server in `../server`, so the whole system deploys as one Azure App Service.

Code is added here next. See `../docs/IMPLEMENTATION-PLAN.md` for the build order.
