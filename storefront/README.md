# Storefront — public gem shop

This folder will hold the public, customer facing gem shop. It is the sales channel: when a customer places an order here, the order is sent to the ERP Sales module through the `/api` endpoints.

The storefront is not the ERP itself. It is one input into it. It is rebranded to Serendib Gems and the design language is shared with the admin app.

The shop is moved in here later in the build.
